# NetacadSearcHelper

# 🚀 Quick Search Pro

**NetacadSearcher** is a Chrome extension that helps you quickly search questions from NetAcad on a reference website without manually switching between multiple tabs.

The extension adds a floating control panel to the page that allows you to:

- 🔎 Search copied questions instantly
- 🌐 Open the reference website quickly
- 🔁 Jump back to the NetAcad tab
- 📌 Use a draggable floating panel
- ⚡ Work efficiently even with many open tabs

---

# 📦 Recommended Extension (Text Selection)

Some learning platforms block text selection or copying.

For the best experience it is recommended to install the following extension that allows selecting and copying text.

### Allow Copy+

🔗 https://chromewebstore.google.com/detail/allow-copy-+/ajhbdcgfhlhhmocddefknjjkejcfpbnj?hl=en

This extension enables:

- Text selection on restricted pages  
- Copying protected content  
- Normal clipboard usage  

Quick Search Pro works best when the question text can be copied.

---

# ⚙️ How It Works

1️⃣ Open the **NetAcad page** that contains the question.

2️⃣ Select the question text.

3️⃣ Copy the text using:
Ctrl + C

4️⃣ Use the **floating panel** added by the extension.

---

# 🧰 Panel Buttons

### 🌐 Open Reference Page
Opens the reference website in a new tab.  
If the page is already open, the extension switches to that tab.

### 🔎 Search
Searches the copied question text on the reference page.

### ✂️ First Words
Searches only the first part of the copied question.  
This helps if the full sentence search does not find a match.

### 🔙 Back
Returns to the NetAcad tab where you started.

---

# 🧠 Typical Workflow

1. Select the question on the NetAcad page  
2. Press **Ctrl + C**  
3. Click **Search** in the floating panel  
4. The extension switches to the reference page  
5. Check the result  
6. Press **Back** to return to NetAcad  

This makes navigation between the question and the reference page much faster.

---

# 🛠 Installation

1. Download or clone the extension files
2. Open Chrome and go to:
      chrome://extensions
3. Enable **Developer Mode**
4. Click **Load unpacked**
5. Select the extension folder
